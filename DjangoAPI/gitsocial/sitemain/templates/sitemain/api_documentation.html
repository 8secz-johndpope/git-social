API Documentation

Lines This Week by a User

URL: git-social.com/api/v1/[owner]/[repo]/user/[username]/lines/["week" or "month"]/

Response:

On Success: {"deletes": 0, "lines": 0, "additions": 0, "success": true}

On Failure: {"success": false, "error": "username not found"}

http://git-social.com/api/v1/ArchiveTeam/ArchiveBot/user/ivan/lines/week
Commits This Week by a User

URL: git-social.com/api/v1/[owner]/[repo]/user/[username]/commits/["week" or "month"]/

Response:

On Success: {"commits": 0, "success": true}

On Failure: {"success": false, "error": "username not found"}

http://git-social.com/api/v1/ArchiveTeam/ArchiveBot/user/ivan/commits/week
Badges Owned By a User

URL: git-social.com/api/v1/[owner]/[repo]/user/[username]/badges/["week" or "month"]/

Response: {"success": true, "badges": [4]}

http://git-social.com/api/v1/ArchiveTeam/ArchiveBot/user/ivan/badges/week
This Week Sticker Image For a User

URL: git-social.com/api/v1/[owner]/[repo]/user/[username]/sticker/["week" or "month"]/

Response: {"success": true, "commits": 0, "additions": 0, "deletes": 0, "time": "2019-02-03T00:00:00", "image":[Image as base64 encded plaintext]}

http://git-social.com/api/v1/ArchiveTeam/ArchiveBot/user/ivan/sticker/week
