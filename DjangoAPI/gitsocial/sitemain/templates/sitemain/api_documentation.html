API Documentation

Lines This Week by a User

URL: git-social.com/api/v1/[owner]/[repo]/user/[username]/lines/week/

Response:

On Success: {"deletes": 0, "lines": 0, "additions": 0, "success": true}

On Failure: {"success": false, "error": "username not found"}

http://git-social.com/api/v1/ArchiveTeam/ArchiveBot/user/ivan/lines/week
Commits This Week by a User

URL: git-social.com/api/v1/[owner]/[repo]/user/[username]/commits/week/

Response:

On Success: {"commits": 0, "success": true}

On Failure: {"success": false, "error": "username not found"}

http://git-social.com/api/v1/ArchiveTeam/ArchiveBot/user/ivan/commits/week
This Week Sticker Image For a User

URL: git-social.com/api/v1/[owner]/[repo]/user/[username]/sticker/week/

Response: Image as base64 encded plaintext

http://git-social.com/api/v1/ArchiveTeam/ArchiveBot/user/ivan/sticker/week
Leaderboard Based on Commits this Week

URL: git-social.com/api/v1/[owner]/[repo]/leaderboard/commits/week/

Response (Sorted by Commits):

{"contributors": [{"commits": 12, "name": "David Yip", "username": "yipdw"}, {"commits": 0, "name": "Matt Iggo", "username": "MattIggo"}, {"commits": 0, "name": "C4K3", "username": "C4K3"}], "success": true}

http://git-social.com/api/v1/ArchiveTeam/ArchiveBot/leaderboard/commits/week
Badge Images

Note:

Badge 01 - 03 are for commits

Badge 04 - 06 are for lines
